# TheSession.TradTuneExplorer Search Database

This repository contains the PostgreSQL database schema used by TradTuneExplorer.

The schema includes support for:

* Tune metadata management
* Recording and album relationships
* ABC notation storage and processing
* Melody fingerprint generation
* Contour and interval analysis
* Fuzzy melody search
* N-gram melody matching
* Edit-distance similarity scoring
* Search-optimized materialized views
* Search ranking and relevance algorithms

The database is designed to support large-scale exploration of traditional music collections, enabling searches by tune name, metadata, recordings, and melodic fragments.

## Contents

This repository currently contains:

* PostgreSQL schema definitions
* Functions and stored procedures
* Views and materialized views
* Triggers and indexes
* Search and melody-matching infrastructure

No tune data, recordings, or other content from The Session are included in this repository.

TradTuneExplorer uses data sourced from The Session. Users wishing to populate the database should obtain data directly from The Session and comply with the applicable licensing terms.
